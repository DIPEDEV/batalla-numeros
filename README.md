# Diseño Base
Aplicación hecha especialmente con simplicidad y competitividad. Uso de Online, Leaderbaord, Practica, Logging (Guardar el nombre) & Dificultades Variadas.

## ¿Cómo Jugar?
Al entrar al menú principal, debes ingresar un nombre de usuario (nickname) en la entrada de texto superior, posteriormente deberas elegir entre 3 opciones:
- 1: Modo Práctica
- 2: Partida Grupal
- 3: Código (Partida ya existente)
